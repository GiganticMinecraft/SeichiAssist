# データベースのマイグレーション

## RDBMS
SeichiAssistはデータベースにMySQLを使っています。

## 注意点
DBの内容は他のサービスから読み取られるため、以下の変更を加えた場合は、サービスのクエリも変更してください。
特に、`playerdata`テーブルの整地量、建築量、ログイン時間に関連するカラムに変更を加えた場合は、サービスのクエリを必ず確認するようにしてください。

* カラムの削除・改名・型の変更
* テーブルの削除・改名
